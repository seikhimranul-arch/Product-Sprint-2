# FintLer: AI Financial Clarity Engine

FintLer turns invisible UPI micro-transactions into clear, actionable insights for Indian millennials. It focuses on eliminating 'salary day anxiety' by delivering instant, AI-powered behavioral feedback—without judgment or friction—directly from users' bank emails. No lending. No ads. No data reselling. 

## Key Features
- Google OAuth onboarding (one-tap sign-in)
- Real-time bank transaction ingestion via Gmail
- 4 AI Insight Cards (Summary, Category Alert, Behavioral Trigger, Recommendation)
- Gamified Spending Personality Badge (shareable)
- Privacy-first: Only parses bank alert emails, discards raw data, lets users disconnect any time

## Tech Stack
- Frontend: React + Vite (Tailwind CSS)
- Backend: Supabase Edge Functions, PostgreSQL, Supabase Auth
- Parsing: Regex + Gemini AI fallback

## Competitive Advantage
- Only solution focusing on behavioral insights ('why', not just 'what')
- No lending or credit product conflicts
- Shareable gamification
- Frictionless auto-data ingestion
